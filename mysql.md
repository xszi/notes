## mysql 使用流程

1. 下载安装

[mysql安装]（https://www.runoob.com/mysql/mysql-install.html）

**注意每次下载需要先安装**

2. 修改密码

```js
mysqladmin -u root -p password 123456 // 会跳出输入旧密码
```

3. 下载安装界面管理工具 MySQL WorkBench

连接数据, 点击Schemas，可以创建库、数据表

**注意左侧下面的TAB： Administration / Schemas**

4. 项目连接数据库
